About

With the proliferation of golf apps in recent years booking golf rounds amongst friends and colleagues has become more complex than not. Search FORE Golf has set out to build the ultimate search engine for the fair-weather golfer enabling them to search and save local courses while simultaneously viewing the current weather forecast without the need for a third party weather application. In the future we hope to include other features such as stat tracking and price-point comparison, as well as access to a national tee time database.


The game of golf is already difficult enough. Allow Search FORE Golf the opportunity to at least make the planning process more user friendly.
